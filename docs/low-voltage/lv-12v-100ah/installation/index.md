# LV-12V-100Ah Installation Guide

## Safety Warning

⚠️ Before installation:
- Professional installation required
- Wear insulating protective equipment
- Ensure power is disconnected
- Read product manual

## Installation Environment

### Indoor Installation
- Well ventilated
- Dry, no corrosive gases
- Temperature: -20°C ~ 45°C
- Avoid direct sunlight

### Mounting Requirements
- Load capacity ≥ 15kg
- Securely fixed
- Maintenance space (front/rear ≥ 50cm)

## Installation Steps

### 1. Unpacking and Inspection

1. Check packaging
2. Confirm model
3. Verify accessories

### 2. Mount Battery

1. Use expansion bolts
2. Keep vertical
3. Verify stability

### 3. Electrical Connection

```
Battery + → Breaker → Load
Battery - → Load
```

### 4. Communication Connection

- RS485: A-A, B-B
- CAN: CAN_H-CAN_H, CAN_L-CAN_L

## FAQ

### Q: How to calculate capacity in parallel?
A: Total capacity = Single unit × Number of units (max 400Ah)

### Q: What breaker size is needed?
A: 100A breaker recommended

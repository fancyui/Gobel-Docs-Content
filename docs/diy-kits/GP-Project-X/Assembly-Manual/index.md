## 安全须知

在组装 GP-Project-X 电池套件之前，请务必仔细阅读并遵守以下安全注意事项。

:::danger 电气安全

- **断电操作**：组装过程中确保所有外部电源断开，避免带电操作。
- **防触电**：电池组装完成后，端子间存在高电压（额定 51.2V），接触前务必确认断路器已断开。
- **绝缘要求**：所有导电部件必须正确安装绝缘材料，避免短路。
:::

:::danger 电池安全
- **短路防护**：电芯和连接片暴露时，严禁使用金属工具同时接触正负极，防止短路产生火花或高温。
- **温度要求**：电芯应在室温环境（10°C ~ 35°C）下组装，避免高温或极寒环境操作。
- **正负极确认**：连接电芯时，务必反复确认正负极方向正确，反接将导致严重损坏。
- **电芯搬运**：单个 314Ah 电芯重量较大，搬运时请使用正确的姿势，必要时寻求帮助，避免受伤。
:::

:::warning 机械安全
- **重物搬运**：电池模组组装完成后整体重量较大，移动时请使用脚轮并确保地面平整。
- **锐边防护**：金属框架、连接片和端子可能存在锐边，操作时建议佩戴防护手套。
- **防夹伤**：调整活动电芯压缩侧板时，注意手指不要夹在侧板与电芯之间。
:::

:::warning 一般安全
- **工作环境**：在干燥、通风良好的室内环境中进行组装，远离易燃物和水源。
- **儿童远离**：组装和测试过程中，请确保儿童远离工作区域。
- **合格工具**：使用绝缘手柄的工具，并确保工具规格与螺丝匹配，避免滑丝。
- **个人防护**：建议佩戴绝缘手套和防护眼镜进行组装操作。
:::

:::note
不正确的组装可能导致电池损坏、起火或人身伤害。如对某些步骤不确定，请联系 Gobel Power 技术支持。
:::

## 产品简介

GP-Project-X 是 Gobel Power 推出的磷酸铁锂低压（51.2V）电池 DIY 套件，适用于各种 280Ah 和 314Ah LiFePO₄ 电芯。采用模块化设计，安装简单迅速。

### 主要特点

- **电芯规格**：使用 314Ah LiFePO₄ 磷酸铁锂电芯
- **连接方式**：1P16S 串联
- **BMS 保护**：200A BMS，带 2A 主动均衡
- **额外保护**：具有额外断路器和熔断器保护
- **电芯布局**：电芯直立摆放
- **外壳选项**：可选透明阻燃塑料外壳

### 适用场景

GP-Project-X 适用于家庭储能、离网太阳能系统等低压大容量储能应用场景。

## 螺丝扭矩要求

为确保各连接部位可靠紧固且不过度拧紧导致损坏，请按照以下扭矩要求操作：

| 螺丝规格 | 扭矩要求 |
| :------: | :------: |
|    M5    |   5N.m   |
|    M6    |   8N.m   |
|    M8    |  15N.m   |
|   M10    | 15-20N.m |

:::caution
使用扭力扳手按上述要求拧紧螺丝。扭矩不足可能导致连接松动、发热；扭矩过大可能损坏螺纹或部件。
:::

## 模块组装

在开始主体组装之前，需要先完成以下模块的预组装。

### 1. 底座总成

将 **底座（[01](#部件01)）** 放置在工作台上，将 **脚轮（[31](#部件31)）** 安装到底座底部的四个角上，确保脚轮方向一致并能正常转动。

![底座总成](images/assem-gp-px-base-full.jpg)

### 2. 外壳前板总成

将 **外壳前板（[04](#部件04)）** 正面朝上放置，把 **JK BMS 4.5英寸屏幕（[30](#部件30)）** 和 **金属按钮开关（[29](#部件29)）** 安装到外壳前板的对应开孔位置上，用螺丝固定牢固。

![外壳前板总成](images/assem-case-front.jpg)

### 3. 负极外壳侧板总成

将一块 **外壳侧板（[05](#部件05)）** 平放，把 **电池负极输出端子（[27](#部件27)）** 安装到侧板对应位置上，用螺丝固定。确认端子朝向正确。

![负极外壳侧板总成](images/assem-case-side-n.jpg)

### 4. 正极外壳侧板总成

将另一块 **外壳侧板（[05](#部件05)）** 平放，把 **电池正极输出端子（[28](#部件28)）** 安装到侧板对应位置上，用螺丝固定。

:::note
负极外壳侧板总成与正极外壳侧板总成为镜像关系，安装时务必确认端子朝向正确，避免后续主体组装时方向错误。
:::

![正极外壳侧板总成](images/assem-case-side-p.jpg)

### 5. 断路器总成

将 **断路器延长插片（[23](#部件23)）** 分别插入 **断路器（[32](#部件32)）** 的四个接线口中，用螺丝固定牢固。确保每个插片与接线口接触良好。

![断路器总成](images/assem-breaker.jpg)

### 6. BMS 总成

将 **BMS固定板（[15](#部件15)）** 作为基础，依次安装以下部件：
- 将 **JK BMS 200A（[09](#部件09)）** 安装到 BMS 固定板上
- 将 **JK BMS 通讯板（[10](#部件10)）** 安装到 BMS 旁边
- 将 **绝缘子（[25](#部件25)）** 安装到固定板对应位置（用于固定熔断器）
- 将 **熔断器（[33](#部件33)）** 安装到绝缘子上
- 将已组装好的 **断路器总成** 安装到 BMS 固定板上
- 用 **BMS-断路器连接片（[16](#部件16)）** 连接 BMS 与断路器

![BMS 总成](images/assem-gp-px-bms-plate.jpg)

### 7. 下层电芯模组总成 (电芯9-16)

将 **金属框架（[02](#部件02)）** 平放在工作台上，按照以下步骤组装下层电芯模组：

a. 观察金属框架，一侧带有固定的侧板。面向金属框架时，**让固定侧板位于左侧**。

b. 将 **电芯底部绝缘膜（[14](#部件14)）** 铺设在金属框架底部。按照示意图放入电芯，并使用 **电芯串联连接片（[18](#部件18)）** 将电芯依次串联连接。

![下层电芯模组顶视图](images/assem-cells-b.jpg)

![电芯排布与连接片连接示意图](images/bottom-frame-diagram.jpg)

:::note
图片中左侧为固定侧板，右侧为活动电芯压缩侧板。
:::

c. 将 **活动电芯压缩侧板（[07](#部件07)）** 安装到金属框架另一侧，通过调整侧板上的 4 颗螺丝来调整对电芯的压缩力度，确保电芯排列紧密但不至于过度挤压。

![调整电芯压缩侧板](images/assem-cells-side.jpg)

d. 将 **电芯金属压条（[24](#部件24)）** 放置在电芯顶部，用螺丝固定在固定侧板和活动电芯压缩侧板上。

:::note
电芯金属压条一端带有长腰孔，该端应朝向活动电芯压缩侧板，以便适应侧板的位置调整。
:::

e. 将 **电压检测PCB板B（[12](#部件12)）** 放置在电芯金属压条上，PCB 板带有端子的一端朝向左侧。然后将电压检测PCB板B上的检测线依次连接到相应的电芯连接片上。

### 8. 上层电芯模组总成 （电芯1-8）

上层电芯模组总成的组装方法与下层电芯模组总成相同。区别在于：
- 电芯连接片的位置布局不同（请参考示意图）
- 使用 **电压检测PCB板A（[11](#部件11)）** 代替电压检测PCB板B

![上层电芯模组顶视图](images/assem-cells-t.jpg)

![电芯排布与连接片连接示意图](images/top-frame-diagram.jpg)

## 主体组装

完成所有模块预组装后，按照以下步骤进行主体组装。

### 1. 放置底座总成

将已组装好的底座总成放置在地面上，检查四个 **脚轮（[31](#部件31)）** 是否均已固定到位，并能正常转动和锁止。

### 2. 安装下层电芯模组

将下层电芯模组总成小心放置到底座上，使用螺丝将电芯模组与底座固定牢固。

![安装下层电芯模组](images/gp-px-step1.jpg)

### 3. 安装上层电芯模组

将上层电芯模组总成放置在下层电芯模组总成上方，**确保上下两层模组的活动电芯压缩侧板位于同一侧**。使用螺丝将上下两个电芯模组固定在一起。

![安装上层电芯模组](images/gp-px-step2.jpg)

### 4. 连接电芯 8 与电芯 9

使用 **电芯8-9连接片（[17](#部件17)）** 连接下层模组顶部的 8 号电芯与上层模组底部的 9 号电芯，完成 16 串电芯的串联连接。

![连接电芯8与9](images/gp-px-step3.jpg)

### 5. 安装 BMS 总成

将 BMS 总成固定在上层电芯模组总成的顶部，**熔断器一侧朝向左侧**。使用螺丝将 BMS 固定板与电芯模组框架固定牢固。

![安装BMS总成](images/gp-px-step4.jpg)

### 6. 连接主电路与检测线

- 使用 **熔断器-电芯总正极连接片（[21](#部件21)）** 连接电芯 16 的正极与 **熔断器（[33](#部件33)）**
- 使用 **BMS-电芯总负极连接片（[20](#部件20)）** 连接电芯 1 的负极与 **JK BMS 200A（[09](#部件09)）**
- 使用通讯线将电压检测PCB板与 BMS 连接

![主电路连接正面](images/gp-px-step5.jpg)

![主电路连接背面](images/gp-px-step5-1.jpg)

### 7. 安装外壳总成

将各外壳总成安装到底座上，**正极侧板在左，负极侧板在右**。将 **JK BMS 4.5英寸屏幕（[30](#部件30)）** 和 **金属按钮开关（[29](#部件29)）** 的线缆连接到 BMS 上。

![安装外壳总成](images/gp-px-step6.jpg)

### 8. 连接输出端子

- 使用 **断路器-电池负极输出端子连接片（[19](#部件19)）** 连接断路器与 **电池负极输出端子（[27](#部件27)）**
- 使用 **熔断器-电池正极输出端子连接片（[22](#部件22)）** 连接熔断器与 **电池正极输出端子（[28](#部件28)）**

![连接输出端子](images/gp-px-step7.jpg)

### 9. 固定外壳侧板

将 **H铝合金插条（[08](#部件08)）** 插入外壳侧板与外壳前后板之间的缝隙处，逐一固定好所有外壳侧板。

![固定外壳侧板](images/gp-px-step8.jpg)

### 10. 安装外壳上盖

将 **外壳上盖（[06](#部件06)）** 放置在电池顶部，确保上盖与外壳四周对齐并平稳放置。

![安装外壳上盖](images/gp-px-step9.jpg)

### 11. 组装完成

检查所有螺丝是否拧紧、所有连接片是否安装到位、所有线缆是否连接正确。确认无误后，GP-Project-X 电池套件组装完毕。

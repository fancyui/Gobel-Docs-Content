# HV-48V-200Ah High Voltage Battery Datasheet

## Product Overview

HV-48V-200Ah is a large-scale commercial energy storage battery system using LiFePO4 technology, suitable for industrial and commercial energy storage applications.

## Key Features

- **Capacity**: 200Ah (9.6kWh)
- **Nominal Voltage**: 48V
- **Cycle Life**: ≥8000 cycles (80% DOD)
- **Efficiency**: ≥95%
- **Communication**: CAN / RS485 / Modbus

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Capacity | 200Ah |
| Nominal Voltage | 48V |
| Voltage Range | 42V ~ 54V |
| Max Charge Current | 100A |
| Max Discharge Current | 100A |
| Peak Power | 10kW (30 seconds) |
| Dimensions | 600 x 450 x 200 mm |
| Weight | Approx. 85kg |

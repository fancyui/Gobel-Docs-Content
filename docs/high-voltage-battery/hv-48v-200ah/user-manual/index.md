
# HV-48V-200Ah User Manual

## System Components

- Battery Modules
- Battery Management System (BMS)
- Cooling System
- Communication Module

## Daily Operation

### Starting the System

1. Close battery breaker
2. Wait for BMS self-test
3. Start inverter
4. Monitor operation status

### Shutting Down

1. Turn off inverter
2. Wait for discharge completion
3. Open battery breaker

## Operating Modes

### Mode 1: Peak Shaving
Charge during off-peak, discharge during peak hours

### Mode 2: Backup Power
Auto-switch to battery when grid fails

### Mode 3: Microgrid
Use with solar and generator

## Troubleshooting

⚠️ Press emergency stop button immediately in case of fault!

| Code | Description | Action |
|------|-------------|--------|
| F01 | Battery overvoltage | Check charging system |
| F02 | Battery undervoltage | Charge to recover |
| F03 | Overcurrent | Check load |
| F04 | Overtemperature | Check cooling |
| F05 | Communication fault | Check wiring |

# HV-48V-200Ah Installation Guide

## Safety Warning

⚠️ High voltage system installation must be performed by professionals:
- High voltage electrical license required
- Wear full insulating protective equipment
- Follow operating procedures strictly
- Ensure proper grounding

## Installation Environment

### Room Requirements
- Dedicated battery room
- Well ventilated
- Temperature: -10°C ~ 50°C
- Humidity < 85% RH
- Floor load capacity ≥ 150kg/m²

### Space Requirements
- Front/rear clearance ≥ 80cm
- Top clearance ≥ 50cm
- Side clearance ≥ 30cm

## Installation Steps

### 1. Foundation Check

1. Check floor levelness
2. Confirm load capacity
3. Verify ventilation system

### 2. Equipment Placement

1. Use forklift for handling
2. Level and position
3. Secure with anchor bolts

### 3. Electrical Connection

⚠️ Disconnect all power before proceeding!

```
Three-phase power → Main breaker → Battery cabinet
```

### 4. Communication Connection

- CAN: 120Ω termination resistor
- RS485: Shielded twisted pair

### 5. Grounding Connection

Grounding resistance < 10Ω

## Parallel Configuration

### Maximum Parallel Units
- Parallel units: Max 10
- Total capacity: 96kWh

## Verification Checklist

| Item | Standard |
|------|----------|
| Insulation resistance | ≥ 100MΩ |
| Grounding resistance | < 10Ω |
| Voltage test | Pass |

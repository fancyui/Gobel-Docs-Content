# LP-21700 Cell Datasheet

## Product Overview

LP-21700 is a high-performance lithium-ion cell using advanced lithium iron phosphate technology, suitable for energy storage systems and power bank applications.

## Key Features

- **Capacity**: 5000mAh
- **Nominal Voltage**: 3.2V
- **Cycle Life**: ≥4000 cycles (80% DOD)
- **Operating Temperature**: -20°C ~ 60°C

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Capacity | 5000mAh |
| Nominal Voltage | 3.2V |
| Charge Voltage | 3.65V |
| Discharge Voltage | 2.5V |
| Max Charge Current | 1C (5A) |
| Max Discharge Current | 2C (10A) |
| Internal Resistance | ≤20mΩ |
| Weight | Approx. 75g |
| Dimensions | 21mm diameter, 70mm height |

## Applications

- Energy Storage Systems (ESS)
- Home Energy Storage
- Power Banks
- Power Tools
- Backup Power

## Safety Precautions

1. Avoid overcharge and overdischarge
2. Prevent short circuits
3. Do not expose to fire
4. Use in well-ventilated areas
5. Use only dedicated chargers

# LP-21700 Installation Guide

## Safety Warning

⚠️ Please read the following safety instructions before installation:
- Installers must have electrical work qualifications
- Wear insulated gloves before operation
- Do not operate in wet or humid environments

## Installation Steps

### 1. Inspect the Battery

1. Check for physical damage
2. Measure voltage (3.0V ~ 3.4V)
3. Verify model specifications

### 2. Prepare Tools

- Insulated screwdriver
- Torque wrench
- Multimeter
- Insulating tape

### 3. Connection Methods

#### Series Connection
```
[+] --- [Cell 1] --- [Cell 2] --- [Cell 3] --- [Cell 4] --- [-]
```

#### Parallel Connection
```
    [Cell 1]
    [Cell 2]  Parallel
    [Cell 3]
```

### 4. Torque Settings

| Connection Type | Torque |
|-----------------|--------|
| Nickel strip welding | 0.5 N·m |
| Screw connection | 1.0 N·m |

### 5. Installation Check

1. Measure total voltage
2. Check connection point temperatures
3. Verify insulation

## FAQ

### Q: What if the battery voltage is too low?
A: Use trickle charge mode to slowly recover voltage.

### Q: Can I mix old and new batteries?
A: Never mix old and new batteries. Use the same model and batch.

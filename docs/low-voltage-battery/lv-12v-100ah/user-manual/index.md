# LV-12V-100Ah User Manual

## First Use

### Charging
1. Charge fully before first use
2. Charging time approx. 3 hours
3. All LEDs light up when fully charged

### Precautions
- Do not interrupt first charge
- Ensure installation is in ventilated area

## Daily Use

### Regular Maintenance
- Check connection terminals monthly
- Keep battery surface clean
- Ensure ventilation holes are clear

### Charging Tips
- Use included charger
- Keep ventilated while charging
- Disconnect after full charge

### Discharging
- Avoid discharging above 80%
- Avoid prolonged high-power use
- Capacity reduces in low temperatures

## LED Indicators

| Status | LED |
|--------|-----|
| Charging | Green flashing |
| Full | Green steady |
| Discharging | Blue flashing |
| Low battery | Red flashing |
| Fault | Red steady |

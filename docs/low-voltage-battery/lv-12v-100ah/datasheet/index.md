# LV-12V-100Ah Low Voltage Battery Datasheet

## Product Overview

LV-12V-100Ah is a home energy storage battery system using LiFePO4 technology, supporting parallel expansion up to 400Ah.

## Key Features

- **Capacity**: 100Ah (1280Wh)
- **Nominal Voltage**: 12.8V
- **Cycle Life**: ≥6000 cycles (80% DOD)
- **Parallel Support**: Up to 4 units
- **Communication**: RS485 / CAN

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Capacity | 100Ah |
| Nominal Voltage | 12.8V |
| Charge Voltage | 14.4V |
| Discharge Voltage Range | 10V ~ 14.4V |
| Max Charge Current | 50A |
| Max Discharge Current | 100A |
| Peak Discharge Current | 150A (5 seconds) |
| Dimensions | 330 x 172 x 215 mm |
| Weight | Approx. 12kg |

- Solar Energy Storage
- Off-grid Power
- Backup Power

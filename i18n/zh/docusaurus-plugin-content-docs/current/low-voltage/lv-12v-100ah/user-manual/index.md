# LV-12V-100Ah 使用手册

## 首次使用

### 充电
1. 首次使用前先充满电
2. 充电时间约 3 小时
3. 充满后 LED 指示灯全亮

### 注意事项
- 首次充电不要中断
- 确保电池安装在通风处

## 日常使用

### 日常维护
- 每月检查一次连接端子
- 保持电池表面清洁
- 检查散热口无堵塞

### 充电建议
- 使用配套充电器
- 充电时保持通风
- 充满后及时断开

### 放电使用
- 避免放电超过 80%
- 高功率设备避免长时间使用
- 低温环境下容量会降低

## LED 指示灯

| 状态 | 指示灯 |
|------|--------|
| 充电中 | 绿色闪烁 |
| 充满 | 绿色常亮 |
| 放电中 | 蓝色闪烁 |
| 低电量 | 红色闪烁 |
| 故障 | 红色常亮 |

## 故障代码

| 代码 | 含义 | 处理方法 |
|------|------|----------|
| E01 | 过充 | 停止充电 |
| E02 | 过放 | 充电恢复 |
| E03 | 过流 | 检查负载 |
| E04 | 高温 | 降温处理 |
| E05 | 短路 | 检查线路 |

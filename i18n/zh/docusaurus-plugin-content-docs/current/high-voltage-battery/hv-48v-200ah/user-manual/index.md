# HV-48V-200Ah 使用手册

## 系统组成

- 电池模组
- 电池管理系统 (BMS)
- 散热系统
- 通讯模块

## 日常操作

### 启动系统

1. 闭合电池断路器
2. 等待 BMS 自检完成
3. 启动逆变器
4. 监控系统运行状态

### 关闭系统

1. 关闭逆变器
2. 等待放电完成
3. 断开电池断路器

## 运行模式

### 模式一: 削峰填谷
在电价低谷时充电，高峰时放电

### 模式二: 备用电源
电网断电时自动切换到电池供电

### 模式三: 微电网
配合太阳能和发电机使用

## 监控

### 本地监控
- LCD 显示屏
- 运行状态指示灯
- 紧急停止按钮

### 远程监控
- 云平台
- 手机 APP
- 短信报警

## 维护计划

| 周期 | 项目 |
|------|------|
| 每月 | 检查连接端子 |
| 每季度 | 清洁散热风扇 |
| 每年 | 专业维护检测 |
| 每2年 | 校准系统参数 |

## 故障处理

⚠️ 发生故障时请立即按下紧急停止按钮！

| 故障代码 | 描述 | 处理 |
|----------|------|------|
| F01 | 电池过压 | 检查充电系统 |
| F02 | 电池欠压 | 充电恢复 |
| F03 | 过流 | 检查负载 |
| F04 | 过温 | 检查散热 |
| F05 | 通讯故障 | 检查线路 |

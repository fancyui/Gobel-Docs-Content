# HV-48V-200Ah 安装指南

## 安全警告

⚠️ 高压系统安装必须由专业人员操作：
- 持有高压电工作业证
- 穿戴全套绝缘防护装备
- 严格遵守操作规程
- 确保接地良好

## 安装环境

### 场地要求
- 专用电池室
- 通风良好
- 温度: -10°C ~ 50°C
- 湿度 < 85% RH
- 地面承重 ≥ 150kg/m²

### 空间要求
- 前后预留 ≥ 80cm
- 顶部预留 ≥ 50cm
- 两侧预留 ≥ 30cm

## 安装步骤

### 1. 基础检查

1. 检查地面平整度
2. 确认承重满足要求
3. 检查通风系统

### 2. 设备就位

1. 使用叉车搬运
2. 调整水平位置
3. 固定地脚螺栓

### 3. 电气连接

⚠️ 断开所有电源后操作！

```
三相电源 → 总断路器 → 电池柜
```

### 4. 通讯连接

- CAN: 终端电阻 120Ω
- RS485: 屏蔽双绞线

### 5. 接地连接

接地电阻 < 10Ω

## 并机配置

### 最大并机数量
- 并机数量: 最多 10 台
- 总容量: 96kWh

### 并机步骤
1. 设置主机地址 (01)
2. 设置从机地址 (02-10)
3. 连接通讯线
4. 上电测试

## 调试

### 首次上电
1. 检查所有连接
2. 测量绝缘电阻
3. 逐级上电
4. 配置参数

## 认证清单

| 项目 | 标准 |
|------|------|
| 绝缘电阻 | ≥ 100MΩ |
| 接地电阻 | < 10Ω |
| 耐压测试 | 通过 |

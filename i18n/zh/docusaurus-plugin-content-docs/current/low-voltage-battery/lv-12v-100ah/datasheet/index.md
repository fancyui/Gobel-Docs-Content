# LV-12V-100Ah 低压电池规格书

## 产品概述

LV-12V-100Ah 是一款家用储能电池系统，采用磷酸铁锂技术，支持并联扩展，最高可达 400Ah。

## 主要特性

- **容量**: 100Ah (1280Wh)
- **标称电压**: 12.8V
- **循环寿命**: ≥6000 次 (80% DOD)
- **并联支持**: 最多 4 台
- **通讯方式**: RS485 / CAN

## 技术规格

| 参数 | 数值 |
|------|------|
| 容量 | 100Ah |
| 标称电压 | 12.8V |
| 充电电压 | 14.4V |
| 放电电压范围 | 10V ~ 14.4V |
| 最大充电电流 | 50A |
| 最大放电电流 | 100A |
| 峰值放电电流 | 150A (5秒) |
| 尺寸 | 330 x 172 x 215 mm |
| 重量 | 约 12kg |

## 应用领域

- 家用储能系统
- 太阳能储能
- 离网供电
- 备用电源
- 通信基站

# LV-12V-100Ah 安装指南

## 安全警告

⚠️ 安装前请务必：
- 由专业人员进行安装
- 佩戴绝缘防护装备
- 确保电源已断开
- 阅读产品说明书

## 安装环境要求

### 室内安装
- 通风良好
- 干燥、无腐蚀性气体
- 温度: -20°C ~ 45°C
- 避免阳光直射

### 支架要求
- 承重能力 ≥ 15kg
- 固定牢固
- 预留维护空间 (前后 ≥ 50cm)

## 安装步骤

### 1. 开箱检查

1. 检查包装外观
2. 确认产品型号
3. 检查配件清单

### 2. 固定电池

1. 使用膨胀螺丝固定
2. 保持垂直安装
3. 确认稳固不晃动

### 3. 电气连接

```
电池 + → 断路器 → 负载
电池 - → 负载
```

### 4. 通讯连接

- RS485: A-A, B-B 连接
- CAN: CAN_H-CAN_H, CAN_L-CAN_L

### 5. 并联安装

1. 先单独调试每台电池
2. 关闭所有电池电源
3. 并联连接后再统一上电
4. 设置主机/从机地址

## 常见问题

### Q: 并联后容量如何计算？
A: 总容量 = 单台容量 × 并联数量 (最大 400Ah)

### Q: 需要配置多大的断路器？
A: 建议 100A 断路器

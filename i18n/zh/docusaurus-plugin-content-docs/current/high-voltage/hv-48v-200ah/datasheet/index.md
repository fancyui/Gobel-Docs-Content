# HV-48V-200Ah 高压电池规格书

## 产品概述

HV-48V-200Ah 是一款大型商业储能电池系统，采用磷酸铁锂技术，适用于工商业储能应用。

## 主要特性

- **容量**: 200Ah (9.6kWh)
- **标称电压**: 48V
- **循环寿命**: ≥8000 次 (80% DOD)
- **效率**: ≥95%
- **通讯方式**: CAN / RS485 / Modbus

## 技术规格

| 参数 | 数值 |
|------|------|
| 容量 | 200Ah |
| 标称电压 | 48V |
| 电压范围 | 42V ~ 54V |
| 最大充电电流 | 100A |
| 最大放电电流 | 100A |
| 峰值功率 | 10kW (30秒) |
| 尺寸 | 600 x 450 x 200 mm |
| 重量 | 约 85kg |

## 应用领域

- 工商业储能
- 峰谷电价套利
- UPS 备用电源
- 微电网
- 数据中心

## 系统优势

- 高能量密度

- 智能电池管理系统

# LP-21700 电芯规格书

## 产品概述

LP-21700 是一款高性能锂离子电芯，采用先进的磷酸铁锂技术，适用于储能系统和移动电源应用。

## 主要特性

- **容量**: 5000mAh
- **标称电压**: 3.2V
- **循环寿命**: ≥4000 次 (80% DOD)
- **工作温度**: -20°C ~ 60°C

## 技术规格

| 参数 | 数值 |
|------|------|
| 容量 | 5000mAh |
| 标称电压 | 3.2V |
| 充电截止电压 | 3.65V |
| 放电截止电压 | 2.5V |
| 最大充电电流 | 1C (5A) |
| 最大放电电流 | 2C (10A) |
| 内阻 | ≤20mΩ |
| 重量 | 约 75g |
| 尺寸 | 直径 21mm, 高度 70mm |

## 应用领域

- 储能系统 (ESS)
- 家用储能电池
- 移动电源
- 电动工具
- 备用电源

## 安全注意事项

1. 禁止过充过放
2. 禁止短路
3. 禁止投入火中
4. 请在通风良好的环境下使用
5. 请使用专用充电器

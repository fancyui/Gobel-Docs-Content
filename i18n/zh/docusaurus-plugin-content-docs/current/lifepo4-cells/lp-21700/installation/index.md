# LP-21700 安装指南

## 安全警告

⚠️ 安装前请务必阅读以下安全须知：
- 安装人员必须具备电工操作资格
- 操作前请佩戴绝缘手套
- 禁止在有水或潮湿的环境下操作

## 安装步骤

### 1. 检查电池

1. 检查电芯外观是否有损坏
2. 测量电压是否正常 (3.0V ~ 3.4V)
3. 确认型号规格匹配

### 2. 准备工具

- 绝缘螺丝刀
- 扭矩扳手
- 万用表
- 绝缘胶带

### 3. 连接方式

#### 串联连接
```
[+] --- [Cell 1] --- [Cell 2] --- [Cell 3] --- [Cell 4] --- [-]
```

#### 并联连接
```
    [Cell 1]
    [Cell 2]  并联
    [Cell 3]
```

### 4. 紧固扭矩

| 连接类型 | 扭矩 |
|----------|------|
| 镍片焊接 | 0.5 N·m |
| 螺丝连接 | 1.0 N·m |

### 5. 安装检查

1. 测量总电压
2. 检查各连接点温度
3. 确认绝缘良好

## 常见问题

### Q: 电池电压过低怎么办？
A: 使用涓流充电模式慢慢恢复电压

### Q: 可以新旧电池混用吗？
A: 禁止混用新旧电池，必须使用同规格同批次的电池

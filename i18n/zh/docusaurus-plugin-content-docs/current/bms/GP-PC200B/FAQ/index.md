# GP-PC200B BMS 常见问题解答 (FAQ)

## 上位机软件支持哪些操作系统？
答：目前仅支持 Windows 操作系统。

### 连接上位机需要使用哪种通讯线？
答：需要使用 RS232-USB 通讯线。每组电池出厂时，我们都会免费随附一根该通讯线。

### 用户可以自行升级 BMS 固件吗？
答：可以。我们将提供最新的固件安装包、升级工具及相应的操作手册。

### BMS 是否支持并联？最多支持多少台并联？
答：支持。系统最高支持 63 台 BMS 并联运行。

### 并联时 BMS 的通信地址如何分配？
答：有两种方式：
1. **手动拨码**：根据规格书中的拨码表，手动设置各台 BMS 的拨码开关以分配地址。
2. **自动拨码**：将所有 BMS 的拨码均设为 `000000`（或初始状态），BMS 将会自动分配通信地址。

### 并联时各台 BMS 之间如何进行物理连接？
答：并联时，使用标准网线将各台 BMS 的 RS485B 和 RS485C 端口进行串行连接（手拉手方式）。
> [!IMPORTANT]
> **注意**：RS485B 为输出口，RS485C 为输入口。连接方式为：第一台的 RS485B 接至第二台的 RS485C，依此类推。

### 如果 BMS 与逆变器通信失败，会产生什么后果？
答：若逆变器设置为“锂电池模式”，则必须与 BMS 建立正常通信方能运行。一旦通信中断或失败，逆变器将触报错警（Error）并停止从电池取电或充电。

### 支持WIFI远程监控和管理吗？
答：支持。通过外接我们提供的独立 WiFi 模块，可在 vrm.gobelpower.com 平台或手机 APP 上，对并联电池系统中的每一台电池进行远程监控与管理。平台与 APP 均可永久免费使用。

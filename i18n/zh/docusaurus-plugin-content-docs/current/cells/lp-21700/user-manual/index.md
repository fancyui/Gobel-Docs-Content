# LP-21700 使用手册

## 日常使用

### 充电

1. 使用专用磷酸铁锂充电器
2. 充电环境温度: 0°C ~ 45°C
3. 充电电流不超过 1C
4. 充满后及时断开充电器

### 放电

1. 放电电流不超过 2C
2. 放电环境温度: -20°C ~ 60°C
3. 避免深度放电 (DOD > 80%)

## 维护保养

### 定期检查

- 每月检查一次电压
- 每月检查一次外观
- 每季度检查一次连接

### 清洁

- 使用干布擦拭表面
- 禁止使用化学清洁剂

## 故障排除

| 故障现象 | 可能原因 | 解决方法 |
|----------|----------|----------|
| 充电发热 | 充电电流过大 | 降低充电电流 |
| 电压下降快 | 容量衰减 | 更换电池 |
| 无法充电 | 保护板故障 | 检查保护电路 |

## 废弃处理

请按照当地法规进行废旧电池回收处理，不要随意丢弃。
